# Catalogador

### Sobre
API para catalogar filmes e series

### Tecnologias e Patterns
* Programação reativa funcional usando webflux
* Reactive MongoDB
* Docker
* Consultar metadados dos files e series usando a api [OMDb API](http://www.omdbapi.com/)
